# 📊 Instagram User Segmentation (Unsupervised Learning)

## 📌 Описание проекта

Этот проект использует **unsupervised learning (обучение без учителя)** для анализа пользователей Instagram.

Модель автоматически делит пользователей на группы (сегменты) на основе:

* активности (Instagram visit score)
* уровня трат (Spending rank)

👉 Важно: в этом методе нет правильных ответов — модель сама находит группы.

---

## 🧠 Используемый метод

В проекте используется алгоритм:

**K-Means Clustering**

Он:

* разбивает данные на группы (кластеры)
* находит центр каждой группы
* относит пользователей к ближайшему центру

├── scaler.v1                         # Сохранённый scaler

---

## 🧠 Как работает система

1. Загружается CSV файл с данными пользователей
2. Данные нормализуются (StandardScaler)
3. Применяется алгоритм K-Means
4. Пользователи делятся на группы
5. Модель сохраняется
6. Новые пользователи могут быть отнесены к одной из групп

## 🚀 Установка и запуск

### 1. Установить зависимости

```
pip install pandas scikit-learn matplotlib joblib streamlit
```

---

### 2. Обучить модель

```
python train.py
```

После этого создадутся файлы:

```
instagram_model.v1
scaler.v1
```

---

### 3. Запуск через консоль

```
python predict.py
```

Введите:

* Visit Score
* Spending Rank

---

### 4. Запуск веб-приложения

```
streamlit run main.py
```

Откроется сайт, где можно:

* увидеть кластеры
* ввести нового пользователя
* получить сегмент

---

## 📊 Входные данные

* Instagram visit score
* Spending_rank (0–100)

---

## 📈 Результат

Модель определяет, к какой группе относится пользователь:

* низкая активность
* высокая активность
* потенциальные покупатели
* и другие сегменты
